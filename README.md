# HTTP Request and Multiprocess
## Duplicate Image Finder Script  

Duplicate image finder script is made using Python.
Script has `HTTP Request` `Multiprocessing` `Pool` `Hash`...

In the script, you can dinamically download images from url using `HTTP Get request`
and it shows you if there is duplicate images.
